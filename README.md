# Portfólio — Matheus Renato

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiência como desenvolvedor Fullstack.

## Sobre o Projeto

Este portfólio foi criado com foco em performance, design moderno e boa experiência do usuário. A aplicação possui uma interface limpa e responsiva, com animações suaves e uma identidade visual consistente.

Nele você encontrará:

- Apresentação profissional (Hero)
- Informações sobre mim
- Experiências e habilidades
- Projetos desenvolvidos
- Formas de contato

## Tecnologias Utilizadas

- Vue.js 3 — utilização da Composition API para melhor organização e escalabilidade
- Vite — ambiente de desenvolvimento rápido e otimizado
- CSS — estilização customizada com foco em responsividade e interface moderna
