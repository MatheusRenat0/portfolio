# Portfólio — Matheus Renato

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e formas de contato como desenvolvedor Fullstack.

## Sobre

Este portfólio foi construído com foco em design moderno e performance. Ele apresenta seções de hero, sobre mim, projetos e contato, com animações fluidas e identidade visual consistente.

## Tecnologias

- **[Vue.js 3](https://vuejs.org/)** — framework JavaScript progressivo com Composition API
- **[Vite](https://vitejs.dev/)** — bundler ultrarrápido para desenvolvimento moderno
